# Dotfiles

Only my dotfiles


## Installation

```sh
make
```

### Requirements
- `rsync` for installation process
- Packages:
    + bash
    + bash-completion
    + git
    + neovim
        * ruff
        * clangd
        * pyright
        * bash-language-server
        * vscode-html-languageserver
        * yaml-language-server
        * lua-language-server
        * tinymist
    + tmux
    + vim

## Features

- Using XDG directories
- Components:
    + bash
